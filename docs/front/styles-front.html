¿Qué es este archivo?

Esta página centraliza el sistema de diseño del módulo Frontend Angular. Contiene todos los tokens de CSS, los componentes visuales y los patrones HTML que los artefactos deben reutilizar para mantener consistencia.

Para crear un nuevo artefacto: parte de ../\_template.html, activa la paleta verde API REST (--layer: #0e7a5a), y copia los componentes que necesites de las secciones de abajo.

TIP

El back-link de cada artefacto debe apuntar a index-front.html para volver a este índice, no a ../index.html.

4 · Código

Bloques de código

Código inline: HttpClient, @Component, inject(). Fondo --surface, color --layer.

Bloques <pre>: fondo oscuro #1c1814, texto claro. Scroll horizontal automático.

// TypeScript — Angular 17+ standalone component
import { Component, inject, signal } from '@angular/core';
import { HttpClient } from '@angular/common/http';

@Component({
  selector: 'app-album-list',
  standalone: true,
  imports: [CommonModule],
  template: `
    @for (album of albums(); track album.id) {
      <div>{{ album.titulo }}</div>
    }
    @empty { <p>Sin álbumes.</p> }
  `
})
export class AlbumListComponent {
  private http = inject(HttpClient);
  albums = signal<Album[]>([]);
}
5 · Navegación por pestañas

Tab navigation

Para tutoriales con múltiples archivos o pasos. Usa .tab-nav + .tab-btn con clase .active y paneles .tab-panel.

// album-list.component.ts
albums = signal<Album[]>([]);
private svc = inject(AlbumService);

ngOnInit() {
  this.svc.getAll().subscribe(data => this.albums.set(data));
}
<!-- album-list.component.html -->
@for (album of albums(); track album.id) {
  <app-album-card [album]="album" />
}
@empty {
  <p>No hay álbumes.</p>
}
// album.service.ts
@Injectable({ providedIn: 'root' })
export class AlbumService {
  private http = inject(HttpClient);
  getAll() { return this.http.get<Album[]>('/api/albums'); }
}

7 · Callouts informativos

Cajas informativas

TIP

Usa inject() en lugar de constructor injection en Angular 17+. Es más limpio con standalone components y no requiere declarar el tipo en el constructor.

INFO

Los standalone components importan sus dependencias directamente en el array imports del decorador @Component, sin necesidad de un NgModule.

OJO

Recuerda agregar provideHttpClient() en app.config.ts para que HttpClient esté disponible. Sin esto, la inyección falla en tiempo de ejecución.

JS helpers (copiar al final de cada artefacto)

<script>
  // Tab navigation
  function switchTab(e, panelId) {
    const card = e.target.closest('.card, .container');
    card.querySelectorAll('.tab-btn').forEach(b => b.classList.remove('active'));
    card.querySelectorAll('.tab-panel').forEach(p => p.classList.remove('active'));
    e.target.classList.add('active');
    document.getElementById(panelId).classList.add('active');
  }

  // Copy command
  function copyCmd(btn) {
    const text = btn.previousElementSibling.textContent;
    navigator.clipboard.writeText(text).then(() => {
      btn.textContent = 'OK ✓'; btn.classList.add('copied');
      setTimeout(() => { btn.textContent = 'COPIAR'; btn.classList.remove('copied'); }, 1800);
    });
  }
</script>
