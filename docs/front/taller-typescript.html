Actividad 2 — Métodos en la clase Person

Un método es una función que pertenece a una clase y puede acceder a sus atributos usando this. Vas a agregar dos métodos a la clase Person.

1
Entiende la estructura de un método

Un método se escribe dentro de la clase, después del constructor, sin la palabra function. Usa this para acceder a los atributos del objeto: 

class Person {
  // ... atributos y constructor ...

  // Método: no necesita la palabra "function"
  fullSummary() {
    return this.name + ' — Código: ' + this.code;
  }
}

// Cómo llamarlo:
console.log(person.fullSummary());
// → "Carlos Rodríguez López — Código: 20221234"
2
Agrega el método fullSummary()

Abre ejemplo-perfil-dinamic.html y busca la clase Person dentro del bloque <script>. Agrega el método resaltado después del constructor, antes del cierre de la clase:

ejemplo-perfil-dinamic.html — clase Person
class Person {
  constructor(name, code, idNumber, age, address, phone, courses) {
    this.name     = name;
    this.code     = code;
    this.idNumber = idNumber;
    this.age      = age;
    this.address  = address;
    this.phone    = phone;
    this.courses  = courses;
  }

  // Devuelve un resumen en una sola línea de texto
  fullSummary() {
    return this.name + ' — Código: ' + this.code
           + ' — Edad: ' + this.age + ' años';
  }

  // Devuelve cuántos cursos tiene el estudiante
  courseCount() {
    return this.courses.length;
  }

}
3

En el bloque de arranque (la llamada a .then()) agrega estas dos líneas de console.log para verificar que los métodos funcionan:

ejemplo-perfil-dinamic.html — bloque de arranque
loadPerson('data/student.json')

    console.log('Resumen:',        person.fullSummary());

Abre la consola (F12) y deberías ver:

Total de cursos: 2

2
Crea una función loadStudents()

Necesitamos una nueva función que cargue el arreglo completo. Observa la diferencia con loadPerson():

loadPerson() — carga UN objeto

const data = await response.json();
// data es { name, code, ... }

return new Person( ... );

loadStudents() — carga un ARREGLO

const data = await response.json();
// data es [ {...}, {...}, {...} ]

return data.map(d => new Person( ... ));

Agrega esta función al bloque <script>, después de loadPerson:

ejemplo-perfil-dinamic.html — nueva función
async function loadStudents(url) {
  const response = await fetch(url);
  if (!response.ok) throw new Error('HTTP ' + response.status);

  const data = await response.json();   // arreglo de objetos planos

  return data.map(d => {
    const courses = d.courses.map(c => new Course(c.name, c.professor));
    return new Person(d.name, d.code, d.idNumber,
                      d.age,  d.address,  d.phone, courses);
  });
}
3
Crea una función renderList()

Esta función recibe el arreglo de objetos Person y construye una tarjeta (card) de Bootstrap para cada estudiante.

Agrega un contenedor vacío en el HTML donde se insertará la lista, fuera del bloque blanco del perfil actual:

ejemplo-perfil-dinamic.html — nuevo contenedor HTML
<div id="student-list" class="mt-4 row g-3">
  <!-- Las tarjetas se insertarán aquí -->
</div>

Ahora agrega la función en el bloque <script>:

ejemplo-perfil-dinamic.html — función renderList
function renderList(students) {
  const container = document.getElementById('student-list');

  container.innerHTML = students.map(s =>
    '<div class="col-12 col-md-4">' +
      '<div class="bg-white border rounded p-3">' +
        '<p class="fw-bold mb-1">' + s.name                      + '</p>' +
        '<p class="text-secondary small mb-2">Código: ' + s.code  + '</p>' +
        '<p class="text-secondary small mb-0">'
          + s.courseCount() + ' cursos matriculados</p>' +
      '</div>' +
    '</div>'
  ).join('');
}
