HTML y CSS

Fundamentos · Desarrollo de Software en Equipos

1.1 Estructura básica

HTML usa marcas (tags) que definen el significado semántico del contenido. Cada marca abre <tag> y cierra </tag>. Las marcas vacías se escriben como <br/> o <img/>.

La estructura de un documento HTML es arborescente: cada elemento es un nodo, y los nodos se contienen unos dentro de otros formando el DOM (Document Object Model).

Código HTML
<!DOCTYPE html>
<html>
  <head>
    <meta charset="UTF-8"/>
    <title>Título de la Página</title>
  </head>
  <body>
    <h1>Encabezado nivel 1</h1>
    <p>Un párrafo de texto.</p>
  </body>
</html>
Árbol DOM
html ├── head │ ├── meta charset="UTF-8" │ └── title "Título de la Página" └── body ├── h1 "Encabezado nivel 1" └── p "Un párrafo de texto."
INFO

<head>: metadata, título de la página, estilos CSS y scripts. Su contenido no es visible en la página.

INFO

<body>: todo el contenido visible — títulos, párrafos, imágenes, tablas, formularios.

Resultado en el navegador

1.2 Valores y atributos

Un elemento HTML tiene un valor (el contenido entre las marcas) y puede tener atributos que configuran su comportamiento. El valor de los atributos siempre va entre comillas.

<a href="https://www.w3schools.com">Enlace a w3schools</a>
   ↑                                               ↑
atributo "href" con su valor          valor del elemento <a>

Tabla

<table> define una tabla. <tr> es una fila (table row). <th> es celda de encabezado y <td> celda de datos.

Código
<table>
  <tr>
    <th>Nombre</th>
    <th>Apellido</th>
    <th>Usuario</th>
  </tr>
  <tr>
    <td>María</td>
    <td>Díaz</td>
    <td>maria.diaz</td>
  </tr>
  <tr>
    <td>Juan</td>
    <td>Castro</td>
    <td>juan.castro</td>
  </tr>
</table>
Resultado (sin CSS aún)
TIP

Sin CSS la tabla casi no tiene estilo visual. Lo verás cambiar radicalmente en la sección de CSS.

Imagen

<img> incrusta una imagen. src es la URL o ruta local. alt es el texto alternativo para accesibilidad. width y height definen el tamaño en píxeles.

Código
<img
  src="https://picsum.photos/400/250"
  alt="Paisaje de ejemplo"
  width="400"
  height="250"
/>
INFO

La barra al final /> cierra la etiqueta img, que es un elemento vacío (no tiene contenido entre marcas).

Resultado

Enlace

<a> crea un enlace. El atributo href define el destino. Con target="_blank" el enlace se abre en una nueva pestaña.

Código
<!-- Enlace en la misma pestaña -->
<a href="https://developer.mozilla.org">
  MDN Web Docs
</a>

<!-- Abre en nueva pestaña -->
<a href="https://www.w3schools.com"
   target="_blank">
  W3Schools ↗
</a>
Resultado

Formulario

<form> agrupa campos de entrada. Cada <input> tiene type (text, email, submit…), name y opcionalmente value como valor por defecto. <br/> inserta un salto de línea.

Código
<form>
  Ciudad:
  <br/><br/>
  <input type="text"
         name="ciudad"
         value="Bogotá">
  <br/><br/>
  Zipcode:
  <br/><br/>
  <input type="text"
         name="zipcode"
         value="1101001">
  <br/><br/>
  <input type="submit"
         value="Enviar">
</form>
Resultado (sin CSS)
OJO

Este formulario muestra la estructura HTML. El comportamiento (qué hacer al hacer click en Enviar) se define con JavaScript o con un framework como Angular.

2.1 Definir los estilos

CSS funciona con reglas. Cada regla tiene un selector (a qué elemento aplica) y un bloque de declaraciones con pares propiedad: valor separados por ;.

selector {
  propiedad: valor;
  otra-propiedad: otro-valor;
}
INFO

Los estilos se pueden definir dentro del HTML en la etiqueta <style> dentro del <head>. La mejor práctica es separarlos en un archivo .css externo y enlazarlo con <link rel="stylesheet" href="style.css">.

2.2 Aplicar los estilos — Selectores

El selector determina a qué elementos HTML aplica cada regla CSS. Los tres tipos fundamentales son:

Selector de elemento

Aplica a todos los elementos con esa etiqueta. No requiere cambios en el HTML.

CSS
/* Aplica a TODAS las tablas */
table {
  border-collapse: collapse;
  width: 100%;
  background: #f0f8ff;
}

th, td {
  border: 2px solid steelblue;
  padding: 8px 12px;
}

th {
  background: steelblue;
  color: white;
}
Resultado

Selector por ID (#)

El selector id se escribe con # en el CSS y se aplica con id="nombre" en el HTML. Identifica un elemento único en la página.

CSS
/* Selector #id — único en la página */
#titulotabla {
  color: darkred;
  font-weight: bold;
  font-family: Times, serif;
  font-size: 14px;
  font-style: italic;
}
HTML
<!-- Solo este th tiene el id -->
<th id="titulotabla">
  Nombre
</th>
Resultado

Selector por clase (.)

El selector class se escribe con . en el CSS y se aplica con class="nombre" en el HTML. Se puede reutilizar en múltiples elementos y se pueden combinar varias clases en un mismo elemento.

CSS
/* Selector .class — reutilizable */
.centrado {
  text-align: center;
}

.colorazul {
  color: steelblue;
  font-weight: bold;
}

p.resaltado {
  background: lightyellow;
  padding: 8px 12px;
  border-radius: 4px;
}
HTML
<!-- Combinando dos clases -->
<p class="centrado colorazul">
  Centrado y azul.
</p>
<p class="resaltado">
  Solo resaltado.
</p>
<p>Sin clases.</p>
Resultado
