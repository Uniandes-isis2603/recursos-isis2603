Guía de Clases Bootstrap

clases usadas en ejemplo-perfil.html · grilla · espaciado · tablas

¿Qué son las clases Bootstrap?

Bootstrap es una librería CSS que expone utilidades como clases HTML. En lugar de escribir CSS propio para cada elemento, se aplican clases predefinidas directamente en el atributo class. El ejemplo ejemplo-perfil.html usa un subconjunto pequeño y enfocado de esas clases: grilla responsiva, espaciado, tipografía, bordes y tablas.

TIP

Abre ejemplo-perfil.html junto a esta guía. Cada clase documentada aquí aparece marcada en el HTML del ejemplo.

1 · Contenedor y fila

La grilla de Bootstrap necesita tres capas: container → row → col-*. Aquí se documentan el contenedor y la fila.

.container max-width: variable por breakpoint
margin: 0 auto
padding: 0 12px
Centra el contenido horizontalmente y limita su ancho máximo según el tamaño de pantalla. En pantallas grandes (~1200 px) el ancho máximo es 1140 px. Usado en: <div class="container py-5">
.row display: flex
flex-wrap: wrap
margin: 0 calc(var(--bs-gutter-x) * -.5)
Crea una fila flex que contiene columnas. Los márgenes negativos compensan el padding interno de cada columna para que los bordes queden alineados. Usado en: <div class="row g-4 align-items-start">
.g-4 --bs-gutter-x: 1.5rem
--bs-gutter-y: 1.5rem
Establece el espacio (gap) entre columnas y filas en 1.5 rem (24 px). La escala va de g-0 (0 px) a g-5 (3 rem). Aplicado junto con .row para separar la columna izquierda de la derecha.
.align-items-start align-items: flex-start
Alinea las columnas al tope de la fila. Sin esta clase, columnas de distinta altura se estiran para ocupar la misma altura (comportamiento por defecto de flex). Evita que la columna corta se expanda verticalmente para igualar a la larga.

2 · Columnas responsivas

Bootstrap divide cada fila en 12 columnas virtuales. La clase col-N indica cuántas de esas 12 ocupa el elemento. El sufijo -md- indica que la regla se aplica solo en pantallas medianas (≥ 768 px).

Pantalla ≥ 768 px (md) — lado a lado
col-md-5
5 / 12 = 41.67 %
col-md-7
7 / 12 = 58.33 %
Pantalla < 768 px (xs/sm) — apiladas
col-12 — 100 %
columna izquierda ocupa todo el ancho
col-12 — 100 %
columna derecha ocupa todo el ancho
.col-12 flex: 0 0 100%
max-width: 100%
El elemento ocupa el 100 % del ancho de la fila. Al combinarlo con col-md-5, esta regla aplica en móvil y el col-md-5 sobreescribe en pantallas ≥ 768 px.
.col-md-5 @media (min-width: 768px) {
flex: 0 0 41.67%
max-width: 41.67%
}
5 de 12 columnas virtuales → 41.67 % del ancho disponible. Aplica solo en pantallas medianas y mayores (≥ 768 px). Usado en la columna izquierda: nombre, avatar y datos personales.
.col-md-7 @media (min-width: 768px) {
flex: 0 0 58.33%
max-width: 58.33%
}
7 de 12 columnas virtuales → 58.33 % del ancho disponible. 5 + 7 = 12, por lo que las dos columnas llenan exactamente la fila. Usado en la columna derecha: tabla de cursos actuales.
NOTA

Los breakpoints de Bootstrap: sm ≥ 576 px · md ≥ 768 px · lg ≥ 992 px · xl ≥ 1200 px · xxl ≥ 1400 px. Sin sufijo (col-12), la regla aplica a todos los tamaños.

3 · Espaciado

Bootstrap define utilidades de espaciado con la forma {propiedad}{lados}-{nivel}. El nivel va de 0 a 5 y cada paso equivale a 0.25 rem (4 px) multiplicado por el nivel.

nivel 0
0 rem
nivel 1
0.25 rem
nivel 2
0.5 rem
nivel 3
1 rem
nivel 4
1.5 rem
nivel 5
3 rem
.py-5 padding-top: 3rem
padding-bottom: 3rem
padding en el eje y (vertical) nivel 5 → 3 rem (48 px). Crea el espacio superior e inferior alrededor de toda la tarjeta del perfil.
.p-4 padding: 1.5rem
padding en los 4 lados, nivel 4 → 1.5 rem (24 px). Da el espacio interior a la tarjeta blanca principal (.bg-white.border.rounded).
.p-3 padding: 1rem
Padding en los 4 lados, nivel 3 → 1 rem (16 px). Usado en las tarjetas interiores "Datos Personales" y "Cursos Actuales".
.mb-0 margin-bottom: 0
Elimina el margen inferior. Útil para anular el margen por defecto de elementos como <p> o <table> cuando están al final de un contenedor.
.mb-2 margin-bottom: 0.5rem
Margen inferior pequeño → 0.5 rem (8 px). Usado bajo el título "Datos Personales".
.mb-3 margin-bottom: 1rem
Margen inferior estándar → 1 rem (16 px). Separa el enlace "Ver guía" del contenido principal.
.mb-4 margin-bottom: 1.5rem
Margen inferior amplio → 1.5 rem (24 px). Separa el título h1 del avatar, y el avatar de la tarjeta de datos.

4 · Tipografía

Clases que modifican el peso, tamaño, color o alineación del texto sin cambiar el elemento HTML.

.fw-bold font-weight: 700
Texto en negrita. Usado en el nombre "Ana María Suarez" (h1) y en los títulos "Datos Personales" / "Cursos Actuales" (p).
.text-secondary color: #6c757d
Aplica el color gris secundario de Bootstrap (#6c757d). Usado en las etiquetas de la tabla ("Código", "Cédula", "Edad"…) y en los datos de los cursos, para dar menor peso visual a información de soporte.
.small font-size: 0.875em
Reduce el tamaño de fuente al 87.5 % del elemento padre. Usado en el enlace "Ver guía de clases Bootstrap" para que sea discreto respecto al contenido principal.
.text-end text-align: right
Alinea el contenido del elemento a la derecha. Posiciona el enlace a esta guía en la esquina superior derecha de la tarjeta.

5 · Bordes y fondo

Utilidades visuales para agregar bordes, esquinas redondeadas y color de fondo.

.bg-white background-color: #fff
Fondo blanco. Contrasta con el fondo gris claro del body (#f0f0f0) para resaltar la tarjeta del perfil.
.border border: 1px solid #dee2e6
Agrega un borde de 1 px en color gris suave en los cuatro lados. Delimita visualmente la tarjeta principal y las tarjetas interiores de datos y cursos.
.rounded border-radius: 0.375rem
Redondea las esquinas (6 px). Se usa junto con .border para dar estilo de "tarjeta" a los contenedores.
.border-top border-top: 1px solid #dee2e6
Agrega borde solo en la parte superior. Usado en cada <tr> de la tabla de datos personales para separar las filas sin usar el modo con bordes por defecto.

6 · Flexbox

Bootstrap expone las propiedades CSS Flexbox como utilidades de clase. En el ejemplo se usan para centrar el avatar horizontalmente.

.d-flex display: flex
Activa el modelo de caja flex en el elemento. Sus hijos directos se convierten en flex items que se pueden alinear y distribuir con utilidades adicionales. Usado en: <div class="d-flex justify-content-center mb-4"> (contenedor del avatar).
.justify-content-center justify-content: center
Centra los hijos a lo largo del eje principal (horizontal por defecto). Junto con .d-flex centra el círculo del avatar dentro de su columna.

7 · Tablas

Bootstrap extiende el elemento <table> HTML con clases que ajustan bordes, densidad y color. Siempre se parte de la clase base .table.

.table width: 100%
border-collapse: collapse
padding en th/td: 0.5rem
Clase base obligatoria. Aplica ancho completo, colapso de bordes y padding uniforme a las celdas. Sin ella las demás clases de tabla no tienen efecto.
.table-borderless td, th { border: none }
Elimina todos los bordes de celdas y filas. Úsala cuando el contenido ya está delimitado por un contenedor con borde, como en "Datos Personales" donde la tarjeta exterior define el límite.
.table-sm td, th { padding: 0.25rem }
Reduce el padding de las celdas a la mitad para lograr una tabla más compacta. Ideal para datos de soporte como la lista de atributos del perfil.
.table-dark background-color: #212529
color: #fff
border-color: #373b3e
Aplica fondo oscuro y texto blanco. En el ejemplo se aplica solo al <thead> de la tabla de cursos para destacar los encabezados "Curso" y "Profesor". Aplicado en: <thead class="table-dark">.
