¿Qué es Bootstrap?

Bootstrap es un framework de código abierto para construir sitios web y aplicaciones web adaptativas (responsives) a partir de HTML, CSS y JavaScript. Fue desarrollado originalmente por un equipo en Twitter y permite que las aplicaciones se vean bien en cualquier dispositivo, especialmente en teléfonos.

Bootstrap contiene estilos predefinidos para botones, tablas, formularios, barras de navegación y componentes avanzados de interfaz.
DOC

Documentación oficial: getbootstrap.com/docs/4.1 — referencia completa de componentes, grilla y utilidades. En ISIS2603 se trabaja con Bootstrap 4.1.

📐

La Grilla Bootstrap

El sistema de grilla de Bootstrap organiza zonas de la pantalla usando contenedores, filas y columnas. Está implementada con Flexbox y es completamente adaptativa. La grilla se compone de un máximo de 12 columnas y tantas filas como se necesiten.

Los contenedores (.container o .container-fluid) son obligatorios cuando se usa el sistema de grilla. 
<div class="container-fluid">
  <div class="row">
    <div class="col">Columna 1</div>
    <div class="col">Columna 2</div>
    <div class="col">Columna 3</div>
  </div>
</div>

Columnas de Distintos Tamaños

Para columnas de distintos tamaños, se indica cuántos espacios (de los 12 disponibles) ocupará cada columna. Por ejemplo, col-6 ocupa la mitad, col-4 un tercio, y col-10 diez doceavos del ancho total.

La suma de los valores de las columnas en una fila debe ser igual a 12 para un diseño perfecto. 
<div class="row">
  <div class="col-2">Pequeña</div>
  <div class="col-8">Grande</div>
  <div class="col-2">Pequeña</div>
</div>
🎨

Estilos Bootstrap

Bootstrap aplica estilos visuales a los elementos HTML mediante clases CSS. Por ejemplo, class="table" mejora el aspecto de una tabla, class="btn btn-success" crea un botón verde, y class="thead-dark" da un encabezado oscuro a una tabla.

Solo incluyendo el link al CSS de Bootstrap en el <head>, todos los estilos quedan disponibles inmediatamente. 
<link rel="stylesheet"
  href="https://stackpath.bootstrapcdn.com/bootstrap/4.1.3/css/bootstrap.min.css">
NPM

En proyectos con Angular o React, Bootstrap no se enlaza desde un CDN — se instala como dependencia con npm install bootstrap y se importa en el archivo de estilos global:

// styles.css (Angular) o index.css (React)
@import 'bootstrap/dist/css/bootstrap.min.css';

La grilla en acción: esta misma página Avanzado

El layout que estás viendo ahora mismo se puede construir con Bootstrap. La barra de navegación usa navbar, el contenido central usa container con una fila centrada mediante justify-content-center, y la columna principal limita el ancho con col-md-10. Las tarjetas usan card con border-start para el borde izquierdo de color.

En Bootstrap, centrar contenido horizontalmente se logra con row + justify-content-center + col-md-N. 
<!-- Barra de navegación (ancho completo) -->
<nav class="navbar navbar-dark bg-dark px-4 d-flex justify-content-between">
  <span class="navbar-brand fw-bold">Bootstrap Básico</span>
  <div class="d-flex gap-3">
    <button class="btn btn-link text-white">Teoría</button>
    <button class="btn btn-link text-secondary">Práctica</button>
  </div>
</nav>

<!-- Contenido centrado con la grilla -->
<div class="container py-5">
  <div class="row justify-content-center">
    <div class="col-12 col-md-10">
      <div class="card shadow-sm border-0 border-start border-4 border-success">
        <div class="card-body p-4">
          <h5 class="card-title">⚡ ¿Qué es Bootstrap?</h5>
          <p class="card-text text-muted">Bootstrap es un framework...</p>
        </div>
      </div>
    </div>
  </div>
</div>

Bootstrap y Flexbox Avanzado

El sistema de grilla de Bootstrap está construido sobre Flexbox: no son tecnologías separadas. Cuando escribes .row, Bootstrap aplica internamente display: flex; flex-wrap: wrap, por lo que obtienes todo el comportamiento de Flexbox automáticamente. Cada columna (.col, .col-4, etc.) es un flex item — Bootstrap traduce el número de columna a flex-basis con porcentajes fijos. Un .col-4 equivale a flex: 0 0 33.33%, y un .col sin número equivale a flex: 1 0 0%, por eso las columnas iguales reparten el espacio automáticamente.

La grilla de 12 columnas es Flexbox con nombres fijos. col-4 = flex-basis 33.33 %, col-6 = 50 %, col-12 = 100 %.
TIP

Además de la grilla, Bootstrap expone Flexbox directamente con clases utilitarias: d-flex, justify-content-between, align-items-center, gap-3. En frameworks como Angular o React estas clases se aplican a cualquier componente para alinear elementos dentro de tarjetas, navbars o botones — sin escribir CSS propio.

PUNTAJEde 6 preguntas
Pregunta 1 de 6
¿Cuántas columnas como máximo soporta la grilla de Bootstrap?
La grilla Bootstrap está basada en 12 columnas. Este número fue elegido porque es divisible por 1, 2, 3, 4, 6 y 12, dando mucha flexibilidad de diseño.
Pregunta 2 de 6
¿Qué clase se usa para un contenedor que ocupa todo el ancho de la pantalla?
container-fluid crea un contenedor de ancho completo (100%). La clase container, en cambio, tiene un ancho máximo que varía según el tamaño de la pantalla.
Pregunta 3 de 6
¿Qué tecnología CSS usa Bootstrap internamente para implementar su grilla?
Bootstrap 4 migró a Flexbox para implementar su grilla, lo que permite un control mucho más preciso del alineamiento y distribución de espacio.
Pregunta 4 de 6
Si una fila tiene tres columnas con clases col-4, col-4 y col-4, ¿qué ocurre?
4+4+4=12, que es exactamente el total de la grilla. El resultado son tres columnas del mismo tamaño que juntas ocupan el 100% del ancho disponible.
Pregunta 5 de 6
¿Qué clase de Bootstrap aplica un estilo visual mejorado a una tabla HTML?
La clase table de Bootstrap transforma una tabla HTML básica en una tabla elegante con bordes, padding y hover states. Es uno de los estilos más simples y usados.
Pregunta 6 de 6
¿Cuál es el propósito de la clase img-fluid en Bootstrap?
img-fluid aplica max-width: 100% y height: auto a la imagen, lo que hace que se redimensione proporcionalmente para caber en su contenedor en cualquier pantalla.
